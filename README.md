# Description
User CRUD and JWT Authentication Implementation

## Installation

```bash
$ npm install
```

## Configurations
```bash
NODE_ENV=development
PORT=3000

DB_STRING=
DB_COLLECTION_NAME=
DB_USERNAME=
DB_PASSWORD=

JWT_TOKEN=
```


## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test
```

## Prettier
```bash
# prettier
$ npm run lint
```
